# メモタグの操作

freee会計APIを使ったメモタグの取得・作成ガイド。

## 概要

メモタグは取引や経費申請などに付与してデータを分類・識別するためのラベルです。
freee-mcp 経由で作成したデータを識別するために「freee-mcp」というメモタグを付与する運用を推奨します。

## 利用可能なパス

| パス | 説明 |
|------|------|
| `/api/1/tags` | メモタグ一覧・作成 |
| `/api/1/tags/{id}` | メモタグ詳細・更新・削除 |

## 使用例

### メモタグ一覧を取得

```
freee_api_get {
  "service": "accounting",
  "path": "/api/1/tags"
}
```

レスポンスの `tags` 配列から目的の `name` を持つメモタグを探し、`id` を取得します。

### メモタグを作成

```
freee_api_post {
  "service": "accounting",
  "path": "/api/1/tags",
  "body": {
    "company_id": 123456,
    "name": "freee-mcp"
  }
}
```

## メモタグ「freee-mcp」を取得または作成する手順

freee-mcp 経由で作成したデータを識別できるようにするための共通手順です。各レシピ（取引・経費申請・請求書・振替伝票・支払依頼など）から参照されます。

1. メモタグ一覧を取得し、`tags` 配列から `name` が `freee-mcp` のエントリの `id` を探す（上記「メモタグ一覧を取得」を参照）
2. 見つからない場合は「freee-mcp」という名前でメモタグを作成して `id` を取得する（上記「メモタグを作成」を参照）
3. 取得した `id` を、データ作成 API の `tag_ids` フィールドに指定する

各APIでの `tag_ids` の指定先は呼び出し側のレシピを参照してください。
